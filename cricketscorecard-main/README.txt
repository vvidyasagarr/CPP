Code compiling and running details:
    compile the program using " g++ code.cpp" then,
    when executed without arguments,the code will give the deatils of
        team1 details before Match
        team2 details before Match
        Match details
        team1 details after Match
        team2 details after Match
        Match statistics

    when executed with "./a.out n" the code will give details of 
        n Matches which include all the details mentioned above for each Match

    In both of the above cases, the code forms a txt file 
      of name "series.txt" which contains the details of match(es)
